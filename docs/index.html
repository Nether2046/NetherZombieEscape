Nether Zombie Escape


β-04

Nether Zombie Escapeは、Minecraft Java Editionでゾンビエスケープが遊べるマルチプレイサーバーです。








ゾンビエスケープって？


参加したプレイヤーの中からランダムで人間チームとゾンビチームに分かれ、それぞれのチームで勝利を勝ち取るために戦うゲームです。

人間は、ゾンビに感染しないよう、ゾンビを銃などの武器で足止めしながらゴールを目指します。
ゾンビは、より多くの人間を感染させるため、銃弾を避けながら人間を追いかけます。








勝利条件


■ 人間側
・最初の人間がゴールに到達後、ステージに設定された一定時間が経過する


■ ゾンビ側
・全ての人間をゾンビに感染させる
・最初のゾンビがゴールに到達する
