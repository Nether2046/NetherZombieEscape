# サバイバルナイフ
移動用の初期装備としてbeta-2で実装された。攻撃用ではなく，マップ探索および移動用に使用するアイテム。特に，銃を装備しながら移動すると速度が低下するのに対し，サバイバルナイフを装備した場合は速度が低下しないので，移動中の装備アイテムとして用いられる。

<br><br><br>

## 仕様

|  各種仕様  |    |
| ---- | ---- |
|  攻撃方法  |  -  |
|  弾数  |  1  |
|  連射機能  |  -  |
|  リロード時間  |  1.2s  |
|  攻撃力  |  -  |
|  ノックバック  |  -  |
|  装備時の減速度  |  0  |
|  ゾンビが復活するまでの時間  |  -  |
