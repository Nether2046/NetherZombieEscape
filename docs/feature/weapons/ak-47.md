# AK-47
Nether Zombie Escapeの最も早い開発段階で導入された銃。プレイヤーが初期段階で手に入れることができる銃。

<br><br><br>

## 仕様

|  各種仕様  |    |
| ---- | ---- |
|  攻撃方法  |  射撃  |
|  弾数  |  40  |
|  連射機能  |  -  |
|  リロード時間  |  3s  |
|  攻撃力  |  20  |
|  ノックバック  |  2  |
|  装備時の減速度  |  3  |
|  ゾンビが復活するまでの時間  |  8s  |
